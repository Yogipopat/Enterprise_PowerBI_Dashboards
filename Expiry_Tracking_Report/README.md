# ⏳ Expiry Tracking Dashboard

## 📌 Overview
This dashboard tracks upcoming expirations of critical documents such as passports, Iqama, and driving licenses.

## 🔍 Key Insights
- Identification of documents nearing expiry
- Prioritization of urgent renewals

## 📊 Features
- Tabular visual with conditional formatting
- Highlighting:
  - Expiring soon
  - Expired
  - Safe

## ⚙️ Techniques Used
- Conditional formatting for urgency visualization
- Date-based calculations using DAX

## 🎯 Business Impact
- Prevents compliance issues
- Ensures timely renewals
- Improves administrative efficiency

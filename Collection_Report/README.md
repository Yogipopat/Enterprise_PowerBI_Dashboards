# 📊 Collection Analysis Dashboard

## 📌 Overview
This dashboard provides insights into financial collections and transaction tracking.

## 🔍 Key Insights
- Total collections and applied amounts
- Transaction volume
- Outstanding balance tracking

## 📊 Features
- KPI Cards:
  - Total Collection
  - Total Applied Amount
  - Total Transactions
  - Balance Amount
- Column Chart: Year-wise collections
- Line Chart: Monthly collection trends
- Table: Client-wise breakdown

## ⚙️ Techniques Used
- DAX measures for financial calculations
- Time-based trend analysis

## 🎯 Business Impact
- Improves cash flow visibility
- Tracks outstanding payments
- Enhances financial monitoring

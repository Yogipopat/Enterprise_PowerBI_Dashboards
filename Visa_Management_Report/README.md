# 🛂 Visa Management Dashboard

## 📌 Overview
This dashboard provides insights into visa allocation, utilization, and compliance tracking.

## 🔍 Key Insights
- Total vs utilized visas
- Expired and under-process visas
- Remaining visa capacity

## 📊 Features
- KPI Cards:
  - Total Visas
  - Utilized Visas
  - Remaining Visas
  - Expired Visas
  - Safe Visas
- Pie Chart: Nationality-wise distribution
- Stacked Bar Chart:
  - Gender distribution by visa status
  - Top 10 visa categories

## ⚙️ Advanced Logic
- DAX measures for:
  - Visa status classification
  - Expiry tracking
  - Utilization metrics

## 🎯 Business Impact
- Ensures visa compliance
- Helps workforce planning
- Identifies risks and bottlenecks

# 💰 Finance Invoicing Dashboard

## 📌 Overview
This dashboard analyzes revenue performance across clients and time periods.

## 🔍 Key Insights
- Total and average revenue trends
- Top-performing clients
- Best revenue-generating months

## 📊 Features
- KPI Cards:
  - Total Revenue
  - Average Monthly Revenue
- Line Chart: Monthly revenue trend
- Bar Chart: Top 7 clients by sales
- Table: Client-wise invoice breakdown

## ⚙️ Advanced Logic
- DAX measures for:
  - Revenue aggregation
  - Top client identification
  - Monthly performance analysis

## 🎯 Business Impact
- Helps in revenue tracking
- Identifies key clients
- Supports strategic decision-making

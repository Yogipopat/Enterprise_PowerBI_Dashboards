# 🚗 Vehicle Management Dashboard

## 📌 Overview
This dashboard provides a comprehensive view of vehicle utilization across departments, enabling operational tracking and resource optimization.

## 🔍 Key Insights
- Total active vehicles currently in use
- Department-wise vehicle distribution
- Top 5 locations with highest vehicle usage
- Vehicle allocation across users and departments

## 📊 Features
- KPI Cards:
  - Total Active Vehicles
  - Total Departments
  - Total Users
- Pie Chart: Vehicle count by owner
- Funnel Chart: Vehicle condition monitoring
- Bar Charts:
  - Top departments by vehicle usage
  - Top 5 areas by vehicle count

## ⚙️ Advanced Logic
- DAX measures used for:
  - Monthly Vehicle Performance Indicator (MVPI)
  - Vehicles nearing expiry within 1 month

## 🎯 Business Impact
- Helps optimize fleet usage
- Improves tracking of vehicle allocation
- Enables proactive maintenance and expiry monitoring
